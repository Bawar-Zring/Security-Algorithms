About Cipher Encryption

🔒
Simple substitution cipher replacing each letter with another letter a fixed number of positions away in the alphabet
🔄
Uses a shift value between 0-25 for the 26 letters of the alphabet
🔑
Easy to use but can be broken with frequency analysis or brute force
⚔️
Named after Julius Caesar who used it to communicate with his generals
🔒
Substitution cipher that maps each character to a different random character
🔄
Creates a random mapping between characters (the "key")
🔑
More secure than Caesar but still vulnerable to frequency analysis
⚔️
Preserves the substitution key for decryption
